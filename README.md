# Orca

My own media player because I don't like everything else

![main player](./assets/screenshots/screenshot01.png)

Features
 - Single playlist
 - Shuffle
 - Autoplay
 - Next/Prev songs
 - Volume control
 - Seeking

Todo-List
- Multiple playlists
- Repeat
- Favourites
- Audio Spectrum
- Song change notifications

Building

```sh
$ ./build.sh -c -b
```

Running

Copy assets folder to bin/
```sh
$ ./orca
```
